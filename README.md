# object-detection
An object detection course that uses machine learning

## Creating and activating a python virtual environment
```
# We may not need to run this
python3 -m venv env

# linux/mac
source env/bin/activate

# windows
.\env\Scripts\activate
```

## Run the shell
```
python3 shell.py
```
